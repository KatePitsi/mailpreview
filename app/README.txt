REQUIREMENTS:
  - Node.js installed (https://nodejs.org)

HOW TO RUN:
  1. Open Terminal (Mac/Linux) or Command Prompt (Windows)
  2. cd into this folder:
       cd path/to/mailscope-app
  3. Start the server:
       node server.js
  4. Open your browser and go to:
       http://localhost:3333

That's it! The app will connect to your Mailtrap account automatically.

HOW TO USE:
  1. Select your sandbox inbox from the dropdown
  2. Paste your email HTML in the editor
  3. Click "Send & Analyze"
  4. View real results in the tabs:
     - Compatibility: real HTML check from Mailtrap (caniemail.com data)
     - Spam Score: real SpamAssassin score
     - Preview: how Mailtrap received and parsed your HTML
     - Source: raw HTML from Mailtrap API
